# RubicsCube

## Controls:
| Key  | Move |
| ------------- | ------------- |
| ↑  | Right side up  |
| ↓  | Right side down  |
| →  | Turn bottom to right |
| ←  | Turn bottom to left |
| w  | Left side up  |
| a  | Left side down  |
| s  | Turn top to right |
| d  | Turn top to left |
| space | Mix Cube |
| z | back |
